## 📚 학습 내용

재즈 음악의 가장 기본이 되는 세 가지를 단계별로 배웁니다.

| 섹션 | 내용 |
|------|------|
| **음계(Scale)** | 장음계, 단음계의 구조와 특징 |
| **음정(Interval)** | 음과 음 사이의 거리 계산 |
| **화음(Chord)** | 3화음과 7화음 이해 |

각 섹션은 독립적으로 학습할 수 있으며, 순서대로 따라가면 기초가 튼튼해집니다.
