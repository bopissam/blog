# 3. 화음(Chord)

두 음 이상이 동시에 울리는 것이 화음입니다.

**화음이란:** 3도 간격으로 음을 쌓은 것

---

## 3화음 (Triad)

### 4가지 종류

| 이름 | 음정 관계 | 표기 | 음 구성 |
|------|----------|------|--------|
| **장3화음** | 장3도 + 완전5도 | CM, C | 밝은 음색 |
| **단3화음** | 단3도 + 완전5도 | Cm | 부드러운 음색 |
| **감3화음** | 단3도 + 감5도 | Cdim, C〇 | 불안정한 음색 |
| **증3화음** | 장3도 + 증5도 | Caug, C+ | 신비로운 음색 |

### 암기 팁

**장3화음을 먼저 외우세요!**

- 장3화음 → 3음을 반음 내리면 = **단3화음**
- 장3화음 → 3, 5음을 반음 내리면 = **감3화음**
- 장3화음 → 5음을 반음 올리면 = **증3화음**

## C Major Scale의 3화음

| 도수 | 화음 | 종류 | 기능 |
|------|------|------|------|
| **I** | C | 장3화음 | 으뜸화음 (안정적) |
| **IIm** | Dm | 단3화음 | |
| **IIIm** | Em | 단3화음 | |
| **IV** | F | 장3화음 | 버금딸림화음 |
| **V** | G | 장3화음 | 딸림화음 (중요!) |
| **VIm** | Am | 단3화음 | |
| **VIIdim** | Bdim | 감3화음 | 불안정 → 해결 필요 |

**모든 키(Key)에서 같은 패턴입니다!**

---

## 7화음 (Seventh Chord)

3도 간격으로 4개 음을 쌓은 것입니다.

- **CM7** = C + E + G + B (장7화음)
- **C7** = C + E + G + B♭ (우월7화음)
- **Cm7** = C + E♭ + G + B♭ (단7화음)
